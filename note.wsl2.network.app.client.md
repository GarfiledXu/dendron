## wsl 网络管理客户端

### 功能模块

1. wsl 发行版本管理
2. 状态信息展示
3. 端口映射管理
4. 局域网跨端集中管理:远程操作、状态同步和批量配置，客户端互通
5. 日志记录

### 技术栈选型

1. 语言
   1. python 跨平台，以及轻量化是有点，但线程模型和io不具备优势
   2. c# .net 也只是跨平台，线程模型和io方面具备优势，并且还有系统集成的优势，轻松调用windows原生API，可靠性和性能更高

### 单人项目的产品规划和发布

初衷：简单入门一下项目管理的基本流程

#### 开发流程和文档指南

主要阶段：

1. 需求分析和产品设计
   1. 功能列表
      1. 核心功能
      2. 添加优先级
   2. 用例说明
      1. 举例典型用户场景

2. 技术调研和选型
   1. 确定技术栈：选型语言，以及框架
   2. 整体架构设计-架构图
   3. 模块架构图：数据流，线程关系，控制流
   4. 技术方案文档
3. 开发阶段
   1. 实现核心模块
   2. 版本控制
   3. 接口文档
   4. 开发日志
   5. troubleshooting
4. 测试阶段
5. 发布与维护
   1. 打包可执行文件以及安装包
   2. 用户手册
   3. 部署说明
   4. 发布版本记录
