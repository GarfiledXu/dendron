# NVS 直径测量工具需求与设计说明书

## 1. 基恩士iv4直径测量功能拆解

* **1.1 功能说明:** 在指定检测区域内提取目标的圆形边缘，计算其直径值，并将结果与设定阈值进行比较，输出 OK / NG 判定结果
  * 工作流程简概：
    * **工具设定阶段:** 选择工具 → 设置检测区域 → 配置检测参数 → 设置判定条件 → 测试验证 → 保存配置
    * **工具运行阶段:** 图像输入 → 应用检测区域 → 边缘提取 → 直径计算 → 阈值判断 → 输出结果

* **1.2 工具设定的核心概念:**
  * 抽取直径（设定选项）
    * **作用：** 在检测到多个候选圆时，决定“最终选哪一个圆对应的直径”
    * * **可选模式：**
      * **最大直径：** 选择检测区域内最大的圆  
      * **最小直径：** 选择检测区域内最小的圆  
      * **指定直径：** 结合其他条件锁定目标圆
    * **使用场景：**
      * 存在多个圆（孔、环、轮廓）时必须配置
  * 抽取直径详细设定-明暗方向（扩展功能）
    * 作用：选择一种提取圆边缘的判定模式
    * 主控方向：由设定的主控图以及框选roi决定的判定方向（具体需与算法核对）
    * 明->暗：算法内建的模式,xxxx
    * 暗->明:算法内建的模式,xxxx
    * 全部:算法内建的模式,xxxx
  * 缩放设定
    * 作用：默认主控图提取的基准直径为100，将其映射到具体的长度单位，作为最终输出
  * 阈值设定
    * 作用：完整描述是，判定okng的直径阈值，上限决定ok允许的最大直径，下限为ok的最小直径，如果开启 缩放设定则是将缩放后结果进行比对

  * 工具特点：
    * 无学习功能，阈值可设置上下限xxx，基准圆的检测筛选方式灵活，xxx
