## concept

1. 劳动纠纷和劳动争议
2. 违法，犯法，违规
3. 劳动仲裁
   1. 劳动仲裁是解决劳动争议的高效途径
4. 法律体系中 法律，行政法规的区别
   1. 劳动合同法实施条例 属于行政法规，而不是法律，（国务院制定），介于法律以下、部门规章以上
   2. 劳动合同法 属于：法律 ，由 全国人大常委会 制定并通过,级别高于行政法规
   3. 本质区别：
      1. 法律 制定：全国人大、全国人大常委会
      2. 行政法规 制定：国务院
   4. 效力层级
      1. 宪法 > 法律 > 行政法规 > 部门规章 > 地方性法规
   5. 底层级 法规的作用？
      1. 对上层级法律法规 做更细节的补充和实施说明
5. 法律体系结构
   1. 宪法：根本法，母法，效力层级最高， 法律 = 宪法 + 普通法律（刑法、民法典、劳动合同法等）
   2. 基本法
   3. 普通法
   4. 刑法，民法典，劳动法：子法，部门法
   5. 《民法典》《刑法》《劳动合同法》《劳动法》《公司法》《治安管理处罚法》都属于子法
6. 民事和刑事
   1. 民事：权利义务的争议，目的：让对方履约、赔偿、给钱 ，结果：赔钱、支付工资、补偿金，面对对象：单位 & 个人之间
   2. 刑事：触犯刑法的犯罪行为，目的：惩罚犯罪人（关、拘、罚），结果：拘留、判刑、罚金，面对对象：国家 vs 犯罪嫌疑人
7. 法律体系中 条例，规定，办法，分别是什么级别
   1. 条例即国务院制定的行政法规，效力仅次于法律
   2. 部门规章（规定、办法）
      1. 人社部的加班、工资、裁员规定
      2. 仲裁委员会办案规则等
8. 什么是行政？在国家体系中同等层级的概念还有哪些？
   1. 行政就是政府管理社会公共事务的活动，属于国家权力的三大板块：立法，司法，行政
   2. 立法：制定法律法规；司法：审判、监督法律实施；行政：管理、执行、实施法律
9. 劳动争议仲裁委员会是什么机构？
   1. 《劳动争议调解仲裁法》第10条：设立劳动争议仲裁委员会，不定级别、不属于行政机关，依法独立办案
   2. 不是行政机构
   3. 原因：不属于政府部门，没有行政级别，不能做行政处罚，不能做行政决定，只能做出民事裁决，也不属于法律系统
   4. 是一个准司法机构
      1. 准司法机构 = 具有部分司法职能，但不属于法院系统的机构
      2. 不是法院，但有一定的司法属性（如裁判、审理、调解、查事实、做裁决等）
      3. 其决定具有法律效力
      4. **劳动人事争议仲裁委员会（劳动仲裁委）**就属于典型的准司法机构
      5. 对比法院：无强制执行的权力，只能算作法律体系的一部分
   5. 由人社局“主管”但不“归属”
      1. 举例：教育局主管学校，但学校不是教育局; 卫健委主管医院，但医院不是卫健委
      2. 人社局主管仲裁委，但仲裁委是独立机构
   6. 仲裁委是如何建立的？是谁“创建”的？
      1. 根据《劳动争议调解仲裁法》：
         1. 市、区人民政府负责设立劳动争议仲裁委员会 ✔
         2. 人社局负责编制和人员管理
         3. 仲裁委主任通常由人社局局长兼任（行政管理职责）
         4. 仲裁员来自：律师、法学专家、人社系统人员等
      2. **仲裁委的“法人地位”**来自政府
      3. **仲裁业务的“准司法权”**来自法律授权
      4. 行政管理由人社局承担
   7. 为什么仲裁委不是行政处罚机构？
      1. 受理劳动争议
      2. 查明事实
      3. 调查证据
      4. 组织庭审
      5. 下达裁决（裁判性文书）
      6. 这完全不是行政处罚（比如罚款、行政处罚决定书），而是司法属性强的活动，所以是“准司法”。
      7. 叫“准”司法，不是真司法
      8. （业务独立）→ 进行准司法裁决
10. 人社局是什么性质的机构？
    1. 行政机关
    2. 代表政府行为
    3. 基于行政法体系
    4. 人社局是仲裁委的行政主管单位
       1. 负责人员管理（编制、工资）
       2. 负责办公保障
       3. 负责政策指导
       4. 但是 仲裁委“业务上独立”，不受人社局干涉

## 如何触发劳动仲裁？

## 仲裁的次数和仲裁后的流向

1. 仲裁委员会的裁决 只有一次、只有一份
   1. 仲裁委只会对同一个劳动争议 作出一次裁决书
   2. 这份裁决书要么是 普通裁决，要么是 终局裁决
   3. 不会有 “仲裁 → 上诉仲裁 → 再裁决第二次” 这种模式。
   4. 重点：仲裁没有“上诉”的概念
2. 收到仲裁裁决后，只有两个方向
   1. 如果你们任何一方不服仲裁裁决 → 去法院“起诉”
   2. 这个叫 对仲裁裁决不服的起诉
   3. 进入的是法院的“民事诉讼程序”
   4. 法院不是重新让仲裁委仲裁，而是法院自己重新审案
   5. 法院不是仲裁的二审，法院是重新审理（相当于一审）

## 普通裁决和终局裁决

1. 普通裁决
   1. 可诉裁决
   2. 绝大多数劳动仲裁案件属于普通裁决
   3. 裁决书送达之日起 15 日内，被申请人或申请人都可以向法院提起诉讼（称为“起诉”或“提起民事诉讼”）。
   4. 如果 15 日双方都不去法院起诉 → 裁决生效。
   5. 生效后：你可以向法院申请强制执行。
   6. 普通裁决不是立即生效，需要等待 15 天。
   7. 法院受理后，仲裁裁决不再作为执行依据，你得进入法院审判流程（至少半年起）
2. 终局裁决（立即生效，不允许去法院起诉）
   1. 少数情形才是终局裁决 （《劳动争议调解仲裁法》第 47 条）
   2. 涉及劳动者请求支付 不超过当地月最低工资标准 12 个月的工资 (3w左右)
   3. 涉及劳动者请求支付 不超过当地月最低工资标准 12 个月的经济补偿金 (3w左右)
   4. 裁决书送达当天立即生效
   5. 收到终局裁决 = 可以立刻申请法院强制执行
   6. 对方不能起诉你，只能向法院申请撤销裁决（且极难成功）

## 仲裁再审（二次仲裁，重新仲裁）

1. 仲裁通常只会进行一次，裁决只会有一份
2. 出现第二次仲裁（重仲裁）只有一种极少数情况
   1. 法院把仲裁裁决“撤销”了
   2. 法院审查仲裁裁决时，如果发现:程序违法 仲裁庭组成人员不合法 仲裁内容超范围 证据未让双方质证 法院可以 撤销仲裁裁决
   3. 撤销后会发生什么？法院 不会自己裁判劳动争议的内容，会撤销仲裁裁决 → 将案件发回仲裁委重新仲裁
3. 这也是对方被告的拖延一种手段

## 仲裁结果后（赢），再起诉

1. 主动法院起诉的好处
   1. 法律程序会变成：法院直接重新审案，不再受仲裁影响，对方失去“申请撤销仲裁裁决延迟履行”的拖延工具
   2. 主动起诉 → 法院判决 → 直接找法院执行，不需要仲裁文书。
   3. 避免“仲裁结果被法院撤销”的风险，坏情况：公司申请撤销仲裁裁决 → 法院撤销 → 你不得不再进行二次仲裁 → 时间拖半年
2. 如果双方都对裁决不服，同时向法院起诉，会发生什么？
   1. 法院不会开两个案子，只会处理一个。
   2. 流程如下：
   3. 法院收到你的起诉
   4. 法院收到公司的起诉（被告“对仲裁裁决不服”）
   5. 法院会将 两个案子合并审理
   6. 法院直接重新审查劳动争议
   7. 仲裁裁决是否正确已经不重要了。
   8. 你们进入法院的一审民事诉讼程序。

## 仲裁流程和时间线

流程图1：

```bash
【阶段 0：准备阶段】
1. 收集证据
   - 劳动合同 / 入职材料
   - 解除劳动合同通知书
   - 工资流水 / 个税申报记录
   - 考勤记录 / 加班证明 / OA 截图或录屏
   - 绩效、沟通记录、工作证明
   - 社保缴纳记录等

2. 准备仲裁申请材料
   - 仲裁申请书
   - 身份证复印件
   - 证据清单
   - 证据材料

----------------------------------------------------

【阶段 1：劳动仲裁（强制前置程序）】

1. 向劳动争议仲裁委员会提交仲裁申请
   ↓
2. 仲裁委受理 → 给你发送《受理通知书》
   ↓
3. 仲裁委向公司（被申请人）送达材料
   - 仲裁申请书副本
   - 受理通知
   - 要求公司提交答辩意见、举证材料

4. 双方交换证据（正式举证质证）
   - 你提交证据
   - 公司提交证据
   - 仲裁委组织双方互相质证（可现场/书面）

5. 仲裁开庭审理
   - 仲裁庭书记员记录
   - 双方陈述事实、理由
   - 法官式仲裁员主持询问、归纳争议点

6. 庭审结束 → 仲裁评议
   ↓
7. 仲裁裁决书（一次性）
   - 普通裁决（常见）
   - 终局裁决（少见，通常小额案件）

----------------------------------------------------

【阶段 2：仲裁裁决后的分支】

        ┌───────────────────────────┐
        │         如果你胜诉（常见）        │
        └───────────────────────────┘

【情况 A：普通裁决】
- 仲裁裁决送达后 → 等待 15 天是否有诉讼期

   分两种情况：

   A1. 公司不服 → 公司 15 天内去法院起诉
       → 进入法院“一审诉讼”
       → 仲裁裁决失效
       → 法院重新审理劳动争议

   A2. 你主动起诉（建议）
       → 直接进入法院一审
       → 仲裁裁决自动失效（不必等待）
       → 法院重新审案
       → 你的控制力更强

   A3. 双方都不去法院（15 天过）
       → 仲裁裁决自动生效
       → 你可以申请法院强制执行

----------------------------------------------------

【情况 B：终局裁决】
- 裁决书当天生效
- 被告不能“对裁决起诉”
- 只能申请撤销裁决（很难成功）

   如果公司不执行：
   → 你可直接申请法院强制执行

----------------------------------------------------

【情况 C：公司拒不执行仲裁】
不论普通裁决 or 终局裁决，只要裁决已生效，
你都可以：

1. 向法院申请强制执行（执行局受理）
2. 法院查封/冻结公司资产
3. 扣划银行账户资金
4. 扣押应收账款、对公账户
5. 将法人或公司列入失信名单

----------------------------------------------------

【阶段 3：法院诉讼阶段（不服仲裁才会进入）】

1. 立案（提交起诉状 + 仲裁文书）
2. 法院受理
3. 双方交换诉讼材料
4. 法院开庭审理
   - 调查事实
   - 审查证据
   - 双方辩论

5. 法院一审判决

        ┌─────────────┐
        │   判决生效（未上诉）  │
        └─────────────┘
             ↓
        → 可直接进入强制执行

        ┌─────────────┐
        │   不服 → 上诉二审    │
        └─────────────┘
             ↓
        二审为终审 → 生效

----------------------------------------------------

【阶段 4：法院强制执行】

你（申请执行人）提交：
- 申请执行书
- 身份证
- 生效法律文书（仲裁裁决 or 法院判决）

法院执行局会：
1. 冻结公司账户
2. 查封车辆、不动产、设备
3. 扣划存款
4. 扣划企业收入
5. 限制高消费
6. 列入失信黑名单

执行完毕 → 结案。

----------------------------------------------------

【最终状态】

案件结束，你获得：
- 违法解除赔偿金（2N 或 N）
- 加班费
- 年休假折现
- 工资差额
- 经济补偿金（如适用）
- 法院强制执行（如需）


```

流程图2：

```bash
【阶段 0：前期准备】

角色：劳动者（你）
动作：准备和整理证据
文件（你持有）：
  - 劳动合同
  - 解除劳动合同通知书
  - 工资银行流水
  - 个税申报记录
  - 考勤、加班、OA录屏
  - 请假记录 & 审批截图
  - 社保记录

-------------------------------------------------------------

【阶段 1：提交仲裁】

1. 你（申请人） → 仲裁委
   动作：提交仲裁申请
   文件：
     - 仲裁申请书（你写）
     - 身份证复印件
     - 证据清单
     - 所有证据材料

2. 仲裁委 → 你
   动作：确认受理
   文件：
     - 《受理通知书》

-------------------------------------------------------------

【阶段 2：仲裁委送达被申请人（公司）】

角色：仲裁委 → 公司（被申请人）
动作：送达材料、要求答辩

文件：
  - 《仲裁申请书副本》（你的申请书复印件）
  - 《应诉通知书》
  - 《举证通知书》（要求公司提交证据）
  - 《答辩通知书》（要求公司提交答辩意见）

公司需要准备：
  - 《答辩状》
  - 公司证据材料（考勤、工资表、制度等）

-------------------------------------------------------------

【阶段 3：证据交换（举证质证）】

组织方：仲裁委  
参与方：你 & 公司

过程：

① 你 → 仲裁委 & 公司  
   文件：
     - 《证据清单》
     - 各项证据复印件、截图、录屏文字说明

② 公司 → 仲裁委 & 你  
   文件：
     - 《答辩状》
     - 公司证据

③ 仲裁委 → 双方  
   动作：组织“质证”
   文件：
     - 《开庭（质证）通知书》或直接当庭质证

-------------------------------------------------------------

【阶段 4：仲裁开庭】

组织方：仲裁委员会  
出席角色：你（申请人）、公司（被申请人）、仲裁员、书记员

仲裁庭动作：
  - 确认争议事实
  - 询问双方
  - 归纳争议焦点
  - 做庭审笔录

文件：
  - 《开庭通知书》
  - 《庭审笔录》（双方签字确认）

你/公司动作：
  - 陈述事实
  - 解释证据
  - 发表意见

-------------------------------------------------------------

【阶段 5：仲裁裁决】

仲裁委 → 你 & 公司  
文件（最重要）：
  - 《仲裁裁决书》（只有一份，不会有第二份）

裁决类型（两种之一）：
  ① 普通裁决（多数情况）
  ② 终局裁决（特定小额案件）

方向：
  仲裁委 → 双方（送达裁决）

-------------------------------------------------------------

【阶段 6：仲裁后的分支】

=========== 普通裁决（最常见） ===========

【6A-1】若公司不服 → 公司 → 法院  
动作：提起“劳动争议诉讼”  
文件：
  - 《民事起诉状》（公司写）
  - 仲裁裁决书复印件

【6A-2】你主动提起诉讼（建议，防止公司拖延）  
你 → 法院  
文件：
  - 《民事起诉状》（你写）
  - 仲裁裁决书复印件

【6A-3】双方都不提起诉讼（超过15天）  
仲裁裁决 → 自动生效  
你 → 法院（执行局）  
文件：
  - 《申请强制执行书》
  - 生效仲裁裁决书
  - 身份证

=========== 终局裁决（较少） ===========

【6B-1】公司只能做一件事：申请撤销裁决（很难成功）  
公司 → 法院  
文件：
  - 《申请撤销仲裁裁决书》

【6B-2】你可以直接申请强制执行  
你 → 法院执行局  
文件：
  - 《申请强制执行书》
  - 终局裁决书

-------------------------------------------------------------

【阶段 7：法院诉讼阶段（若进入）】

法院动作：
  - 登记立案
  - 送达诉讼材料给对方

文件：
  - 《受理通知书》
  - 《举证通知书》
  - 《开庭传票》

你/公司动作：
  - 提交证据
  - 提交代理词
  - 出庭开庭

法院文件：
  - 《民事判决书》（一审）

若一审双方不服：
  - 15 天内可向中级法院 → 《上诉状》
若无人上诉：
  - 判决生效

你 → 法院执行局  
文件：
  - 《申请强制执行书》
  - 生效文书（仲裁 or 判决）
  - 身份证

法院执行局动作：
  - 查询公司账户
  - 冻结、扣划资产
  - 查封房产、车辆、设备
  - 拍卖资产
  - 列入失信名单

公司（被执行人）收到文件：
  - 《执行通知书》
  - 《财产申报令》
  - 《限制高消费令》（若拒不履行）

最终：
  → 执行完毕 → 结案
```
